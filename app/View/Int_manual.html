Manual de Usuario - Sistema 3Shape

Introducción

Bienvenido al sistema de gestión de activos y equipos 3Shape, una plataforma integral diseñada para optimizar el control y seguimiento de los equipos en su empresa. Este sistema proporciona herramientas avanzadas para una gestión eficiente de los activos empresariales, permitiendo un control preciso del inventario, seguimiento de movimientos y mantenimiento preventivo.

Características principales del sistema:

  • Interfaz intuitiva y fácil de usar
  • Gestión centralizada de activos
  • Sistema de notificaciones en tiempo real
  • Generación automática de reportes
  • Seguimiento completo del ciclo de vida del equipo

Gestión de Activos

Control de Inventario

El sistema permite una gestión completa del inventario:

  • Registro detallado de activos con información técnica completa (modelo, número de serie, especificaciones)
  • Seguimiento en tiempo real del estado de cada equipo (disponible, en uso, en mantenimiento)
  • Historial completo de movimientos y mantenimiento con fechas y responsables
  • Búsqueda avanzada con múltiples criterios (ubicación, estado, usuario asignado)
  • Categorización flexible de equipos por tipo, departamento o proyecto
  • Gestión de garantías y fechas de mantenimiento programado
  • Reportes personalizables para análisis de inventario

Registro de Entrada y Salida

Control de Movimientos

Características principales del control de movimientos:

  • Registros detallados de entrada y salida de equipos con fecha
  • Sistema de firma digital para validación de movimientos con seguridad biométrica
  • Generación automática de certificados y documentos de soporte en formato PDF
  • Próximamente: Registro fotográfico del estado del equipo en cada movimiento

Sistema de Notificaciones

Alertas y Monitoreo

Características del sistema de notificaciones:

  • Alertas de mantenimiento programado
  • Notificaciones de movimientos de equipos
  • Registro de actividad del sistema

Funciones Administrativas

Gestión de Campos Dinámicos

Características avanzadas de personalización de base de datos:

  • Creación dinámica de campos para tablas de activos y usuarios
  • Selección de tipo de campo personalizado (texto, número, fecha, etc.)
  • Configuración de reglas de validación de campos
  • Actualizaciones en tiempo real de formularios sin reinicio del sistema
  • Sincronización automática de la estructura de la base de datos
  • Gestión de dependencias de campos

Administración de Base de Datos

Herramientas completas para la base de datos:

  • Funcionalidad de respaldo y restauración de base de datos
  • Herramientas para modificar la estructura de tablas
  • Importación/exportación de datos en múltiples formatos
  • Verificaciones automáticas de validación de datos
  • Utilidades para optimización de base de datos

Panel de Control y Analíticas

Características de monitoreo estadístico:

  • Gráficos de distribución del estado de los equipos
  • Análisis de los principales cargos laborales para distribución de empleados
  • Visualización del seguimiento de usuarios registrados por rol
  • Monitoreo y actualizaciones de datos en tiempo real

Mantenimiento del Sistema

Herramientas administrativas de mantenimiento:

  • Gestión y monitoreo de archivos de registro
  • Configuración de notificaciones del sistema
  • Control y configuración de acceso de usuarios
  • Modificaciones en la estructura de la base de datos
Información de Control de Inventario

El objetivo principal de este servicio es optimizar la gestión de datos de activos para los usuarios. Proporciona funcionalidades detalladas que cubren el ciclo de vida del equipo, tales como: Registro de nuevos activos. Actualización de información de activos. Gestión de salidas de equipos (asignaciones) y cambios/intercambios. Eliminación de activos obsoletos o dados de baja. Además, incluye validación en tiempo real para verificar la existencia del activo y la unicidad del número de serie del equipo.

Información de Gestión de Campos Dinámicos

El sistema de campos dinámicos es una característica poderosa diseñada para mejorar la mantenibilidad del software y proporcionar a los administradores un mayor control sobre la aplicación. Este sistema permite:

  • Adición de nuevos campos personalizados a la base de datos, mejorando la recopilación de datos y la experiencia del usuario
  • Gestión flexible de campos con opciones para agregar o eliminar columnas según sea necesario
  • Sistema avanzado de administración de datos para la visualización de la tabla principal
  • Capacidades dinámicas de inserción y eliminación de datos (requiere sesión activa de administrador)
  • Funcionalidad de exportación de base de datos en formatos CSV o TXT
  • Gestión dinámica completa de datos que permite la integración de campos con los formularios correspondientes
  • Actualizaciones en tiempo real de formularios tanto para procesos de registro como de modificación

Información de Panel de Control y Analíticas

El panel analítico integra múltiples fuentes de datos en tiempo real:

  • Visualización del estado del equipo a través de registros de actividad
  • Análisis geográfico usando datos de ubicación de la tabla de activos
  • Seguimiento de garantías vinculado a la tabla de registros de mantenimiento
  • Generación de reportes personalizados (PDF/CSV) con historial de registros
  • Datos actualizados cada 5 minutos desde la base de datos principal
  • Integración con herramientas existentes de administración de base de datos

Los reportes usan los mismos mecanismos de generación de PDF que el módulo de movimientos (ver Int_LogsDelete.php)
