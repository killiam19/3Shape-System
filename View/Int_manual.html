User Manual - 3Shape System

Introduction

Welcome to the 3Shape asset and equipment management system, a comprehensive platform designed to optimize the control and tracking of equipment in your company. This system provides advanced tools for efficient enterprise asset management, allowing precise inventory control, movement tracking, and preventive maintenance.

Main features of the system:

  • Intuitive and easy-to-use interface
  • Centralized asset management
  • Real-time notification system
  • Automatic report generation
  • Complete equipment lifecycle tracking

Asset Management

Inventory Control

The system allows complete inventory management:

  • Detailed asset registration with complete technical information (model, serial number, specifications)
  • Real-time tracking of each equipment's status (available, in use, under maintenance)
  • Complete history of movements and maintenance with dates and responsible parties
  • Advanced search with multiple criteria (location, status, assigned user)
  • Flexible equipment categorization by type, department, or project
  • Warranty management and scheduled maintenance dates
  • Customizable reports for inventory analysis

Movement Control

Inventory Control Information

The main objective of this service is to optimize asset data management for users. It provides detailed functionalities covering the equipment lifecycle, such as: Registering new assets. Updating asset information. Managing equipment check-outs (assignments) and changes/swaps. Deleting obsolete or decommissioned assets. Additionally, it includes real-time validation to verify asset existence and the uniqueness of the equipment's serial number.

Movement Control Information

The main objective of this service is to retrieve specific asset information from the database, identifying them by their 'assetname', in order to generate documents or reports in PDF format. Additionally, it offers key functionalities such as: Digital Signature: Allows adding a digital signature to the selected PDF documents before their final generation or distribution. Asset Management (Separate Functionality): The overall system also includes options to modify an asset's information or to decommission it (remove it from the system)."

Dynamic Fields Management Information

The dynamic fields system is a powerful feature designed to enhance software maintainability and provide administrators with greater control over the application. This system enables:

  • Addition of new custom fields to the database, enhancing data collection and user experience
  • Flexible field management with options to add or remove columns as needed
  • Advanced data administration system for the main table display
  • Dynamic data insertion and deletion capabilities (requires active administrator session)
  • Database export functionality in CSV or TXT formats
  • Comprehensive dynamic data management allowing field integration with corresponding forms
  • Real-time form updates for both registration and modification processes

Dashboard and Analytics Information

The analytical dashboard integrates multiple real-time data sources:

  • Equipment status visualization through activity logs
  • Geographic analysis using location data from the assets table
  • Warranty tracking linked to the maintenance_records table
  • Custom report generation (PDF/CSV) with log history
  • Data updated every 5 minutes from the main database
  • Integration with existing database administration tools

Reports use the same PDF generation mechanisms as the movements module (see Int_LogsDelete.php)
